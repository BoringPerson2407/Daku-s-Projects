Semantic tags summary

What are Semantic tags?

Semantic tags in html provides meaning to a web content. Instead of using generic <div> and <span>, semantic tags decribe the purpose of the content, improving readability, accessibility, and SEO.

For example <header> clearly indicates that it contains the website's header, while <article> defines an independent content section.

Why use semantic tags?

  • Improves readability
  • SEO benefits
  • Accessibility
  • Better Maintainability

Commonly used tags

Tag Description
<header> Represents the top section of a page or section.
<nav> Defines navigation links (menus)
<section> Groups related content together
<article> Represents self-contained content (like blog posts)
<aside> Used for side content (like ads or sidebars)
<footer> Represents the bottom section of a page
Tag Description
<main> Contains the main content of a webpage
<figure> Groups images or graphics with captions
<figcaption> Provides a caption for a <figure>
<time> Represents dates and times
<mark> Highlights important text
<address> Defines contact details
